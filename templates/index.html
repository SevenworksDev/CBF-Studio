Ever wanted to build a simple Geometry Dash bot for free without decent knowledge on coding?

You're in luck!

How does it work?

Using the Dashboard, you can manage all your Geometry Dash bots. To create a new bot, you can click the "Create Bot" button which pops up a form (or select a template).
After submitting, you are brought to the Bot Editor where you can edit commands and other information and hit "Build" to create a .ZIP containing all the files needed to run your new bot you just created!
Submitting "Bot Account" details is safe and secure, These details are only used for turning a CBF file into a .zip containing cbf.config (where info is stored), "run.bat" (for running the bot easily) and the CBF Runner (cbf.exe). Zip Builds are only hosted for a tiny amount of time until deletion to confirm that your build is downloaded successfully.
DON'T BE STUPID! Your bot account needs to be a fresh account, CBF Studio is 100% bannable by Geometry Dash moderation (for some reason)
